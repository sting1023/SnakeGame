# SnakeGame PRD

## 项目概述
- 项目名称：SnakeGame（贪吃蛇）
- 一句话描述：在 Android 手机上运行的经典贪吃蛇游戏
- 目标用户：休闲游戏玩家
- 核心价值：简单、好玩、快速上手

## 功能需求
### 必备功能 (MVP)
- 贪吃蛇移动（方向控制）
- 食物生成与吃食物
- 得分显示
- 游戏结束判定（撞墙、撞自己）
- 重新开始
- 触摸滑动控制 + 按钮控制

## 技术选型
- 语言：Kotlin
- UI：Android Views（Canvas 绘制）
- 构建：Gradle + GitHub Actions
- 签名：JKS keystore
